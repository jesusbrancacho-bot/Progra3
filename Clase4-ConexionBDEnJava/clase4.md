# Computación en la Nube

- La computación en la nube es la entrega bajo demanda de potencia de cómputo, bases de datos, almacenamiento y otros servicios a través de Internet.

- El modelo de precios de pago por uso elimina la necesidad de realizar inversiones iniciales en infraestructura física.

- La agilidad permite reducir el tiempo de provisión de recursos tecnológicos de semanas a solo instantes.

- La elasticidad facilita el escalado automático de los recursos para coincidir exactamente con la demanda fluctuante.

- El ahorro de costos se obtiene al intercambiar gastos de capital fijos por gastos operativos variables.

# Amazon Web Services (AWS)

- AWS es una plataforma que ofrece numerosos servicios integrales desde centros de datos distribuidos globalmente.

- La infraestructura se organiza en Regiones geográficas y Zonas de Disponibilidad con redundancia física.

- Proporciona una consola de administración y APIs para el control total de los recursos de forma programática.

- Soporta millones de clientes activos, incluyendo startups, grandes empresas y agencias gubernamentales.

- Ofrece el cumplimiento de estándares internacionales de seguridad y auditoría de forma nativa.

# Servicios AWS

- Amazon RDS facilita la configuración, el funcionamiento y el escalado de bases de datos relacionales administradas.

- Amazon RDS soporta motores de bases de datos como MySQL, PostgreSQL, Oracle y SQL Server.

- Amazon EC2 proporciona capacidad de cómputo segura y redimensionable mediante instancias de servidores virtuales.

- Amazon S3 es un servicio de almacenamiento de objetos diseñado para ofrecer alta disponibilidad y durabilidad.

# AWS Academy

- AWS Academy proporciona a instituciones de educación superior un currículo de computación en la nube gratuito.

- El contenido está desarrollado por expertos técnicos de AWS y se mantiene actualizado según la evolución de la plataforma.

- Los laboratorios prácticos ofrecen acceso a la consola real de AWS para desarrollar habilidades técnicas.

- El programa prepara a los estudiantes para obtener certificaciones oficiales reconocidas por la industria.
